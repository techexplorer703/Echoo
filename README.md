# Echooo 🧠

Echooo is an AI-powered Personal Memory & Knowledge Management Platform designed to become a user's second brain.

## Vision

Build an intelligent assistant that can:

- 🧠 Store memories
- 💬 Chat with AI
- 📚 Organize knowledge
- 🔍 Search everything instantly
- 📄 Manage documents
- 📅 Remember events and timelines

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

### Backend (Coming Soon)
- FastAPI
- PostgreSQL
- Redis
- Vector Database
- AI APIs (OpenAI / Gemini)
